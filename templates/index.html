Do You Know Ball API

Endpoint Reference · v1.0

Authentication

All endpoints require an API key passed as a request header.

X-API-Key: your_api_key_here

Don't have an API key? See /license for instructions on how to register and generate one.

Get Player Stats

Retrieve stats for a single player. Optionally filter by dataset or specific stat columns.

GET/player
ParameterDefaultDescription
namerequired — Full player name (e.g. Juan Soto)
datasetoptional 2025 One of 2025, 3year, or projections
statsoptional all Comma-separated columns (e.g. AB,R,H,1B,2B,3B,HR,RBI,BB,K,SB,CS,AVG,OBP,SLG,FPTS). Returns all if omitted. 
curl "https://do-u-know-ball.com/player?name=Juan%20Soto" \
  -H "X-API-Key: your_api_key_here"

curl "https://do-u-know-ball.com/player?name=Juan%20Soto&dataset=3year" \
  -H "X-API-Key: your_api_key_here"

curl "https://do-u-know-ball.com/player?name=Juan%20Soto&stats=HR,AVG,FPTS" \
  -H "X-API-Key: your_api_key_here"
{
  "Player":   "Juan Soto",
  "Position": "OF",
  "Team":     "NYM",
  "AB":       577,
  "HR":       43,
  "AVG":      0.263,
  "FPTS":     911
}

List All Players

Returns a list of every available player name in the 2025 dataset.

GET/players
curl "https://do-u-know-ball.com/players" \
  -H "X-API-Key: your_api_key_here"
[
  "Shohei Ohtani",
  "Juan Soto",
  "Aaron Judge",
  "Mookie Betts",
  ...
]
